Função e Função Afim

Função:

Uma função é uma relação matemática que associa cada elemento de um conjunto de partida, chamado domínio, a um único elemento de um conjunto de chegada, chamado contradomínio. Em outras palavras, uma função atribui um valor de saída para cada valor de entrada.Existe, portanto, uma relação de dependência entre essas grandezas. Essas relações são expressas por fórmulas que no meio matemático recebem o nome de funções.

Função Afim:

Uma função afim, também conhecida como função linear, é um tipo específico de função em que a relação entre as variáveis é descrita por uma equação linear. A forma geral de uma função afim é dada por f(x) = ax + b, onde "a" e "b" são constantes e "x" é a variável de entrada.

Domínio:

O domínio de uma função é o conjunto de todos os valores válidos que podem ser usados como entrada para a função. É o conjunto de valores para os quais a função está definida e produz um resultado significativo. Por exemplo, em uma função que calcula a raiz quadrada de um número real, o domínio seria o conjunto de todos os números reais não negativos, pois a raiz quadrada de um número negativo não é um número real.

Imagem:

A imagem de uma função é o conjunto de todos os valores que a função pode assumir como resultado, considerando todos os elementos do domínio. Em outras palavras, a imagem é o conjunto de todos os valores possíveis que a função pode produzir. A imagem também é conhecida como o contradomínio da função.
